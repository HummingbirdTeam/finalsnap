FinalSnap Privacy Policy

Last updated: May 10, 2026

FinalSnap is a Chrome extension for capturing full-page screenshots, editing them locally, and exporting images or PDFs. This policy explains what data the extension processes and how that data is handled.

Data Processed Locally

FinalSnap processes screenshot and editor data locally in the user's browser. This may include:

  • Screenshot image data from the active tab after the user starts a capture.
  • Local screenshot history saved in the user's Chrome profile.
  • Editor actions, annotations, crop settings, erase areas, and export settings.
  • Extension preferences such as language, default image format, and PDF settings.

Screenshot image data is not uploaded to FinalSnap servers by the extension.

Data Sent to a License or Promotion Server

If Pro licensing, paid features, beta access, or promotion-based Pro rewards are enabled, FinalSnap may contact a license or promotion server to verify access. The extension may send:

  • An installation ID generated for license and trial tracking.
  • An optional email address if the user provides one for license lookup, beta access, support, or Pro code delivery.
  • The extension version.
  • Trial status, remaining Pro trial days, and limited feature-use counters.
  • Payment or subscription identifiers returned by the payment provider.
  • Promotion claim details submitted by the user, such as the promotion platform and public post URL.

This data is used only to verify Pro access, manage purchases, support subscriptions, review promotion reward claims, provide beta or support access, and prevent abuse.

Support and Feedback

If a user chooses to send a support or beta-request email from FinalSnap, the email may be prefilled with diagnostic context such as the extension version, installation ID, page title or URL related to the issue, basic extension settings, browser language, platform, and browser user agent. FinalSnap does not automatically attach screenshots or upload screenshot image data.

Payments

Payments, if offered, are processed by Stripe Checkout or another disclosed payment provider. FinalSnap does not collect, process, or store credit card numbers.

Chrome Permissions

FinalSnap requests Chrome permissions only to provide its screenshot, editing, export, source display, and local history features. For example, it uses access to the active tab only after the user starts a capture, uses downloads to save exported files, uses storage to keep local settings and screenshot history, and uses Chrome's favicon permission to show the captured page's site icon inside the optional browser-style source bar.

Data Sharing

FinalSnap does not sell personal data. Screenshot content is not shared with third parties by the extension. Payment information is handled by the payment provider when a user chooses to purchase paid features.

FinalSnap's use and transfer of information received from Google APIs follows the Chrome Web Store User Data Policy, including the Limited Use requirements.

Data Retention and Deletion

Screenshot history and extension settings remain in the user's Chrome profile until the user deletes them, clears browser extension data, or uninstalls the extension. Users may delete local screenshot history from the extension interface when that feature is available.

Children's Privacy

FinalSnap is not directed to children under 13 and does not knowingly collect personal information from children.

Changes to This Policy

This privacy policy may be updated from time to time. The updated version will be posted on this page with a new "Last updated" date.
