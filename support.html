Support

FinalSnap helps you capture, restructure, clean, and export long web screenshots. If something does not work as expected, send a concise report and we will use it to improve the product.

Email support

Contact finalsnapapp@gmail.com for product support, bug reports, Chrome Web Store review questions, beta access, tester access, or Pro access questions.

What to include

  • FinalSnap version.
  • Chrome version and operating system.
  • The page URL you were trying to capture, if it is safe to share.
  • What you expected to happen.
  • What actually happened.
  • Your browser authorization ID if you are asking about tester, beta, trial, or Pro access.

Privacy note

FinalSnap's report buttons may prefill useful diagnostic context, but they do not automatically attach or upload screenshot image data. You can review and edit the email before sending it.

Common first checks

  1. Refresh the target page and capture again.
  2. Avoid Chrome internal pages such as chrome://, where Chrome blocks capture scripts.
  3. For very long pages, try exporting after a quick crop or Cut Middle cleanup.
  4. If a page has popups, ads, or lazy loading, close visible overlays before capture when possible.
